# Online-book-buying-and-selling-portal-
An Online Book Buying and Selling Portal developed using *Java* that allows users to buy, sell, and manage books through a web-based platform. The system provides separate roles for users and administrators to ensure secure and efficient book transactions.

---

## Features

### User Features
- User registration and login
- Browse available books
- Search books by title, author, or category
- Buy books online
- View order history
- Manage user profile

### Admin Features
- Admin login
- Add new books
- Update book details
- Delete books
- Manage users
- View all orders

---

## Technologies Used

- *Programming Language:* Java
- *Frontend:* HTML, CSS, JavaScript
- *Backend:* Java (Servlets / JSP)
- *Database:* MySQL
- *Server:* Apache Tomcat
- *IDE:* Eclipse / IntelliJ IDEA
- *JDBC:* For database connectivity

---

## System Requirements

- JDK 8 or above
- Apache Tomcat Server
- MySQL Server
- Web Browser (Chrome / Firefox)
